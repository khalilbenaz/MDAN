## Système de Mémoire

```

├── core/config.yaml            ← Configuration dial le projet
└── _config/manifest.yaml       ← État dial l'installation

Le contexte kaybqa entre :
- Les wizards (le PRD 3endo accès l le brief)
```

## Debate Protocol

Mlli katwsel décision critique (choix de stack, pattern, priorisation), les agents kaydébatiw :

```
Round 1: Ouverture     → Kol agent kayprésenté la position dyalo
Round 2: Challenge     → Kol agent kayattaqué la position dial l'autre
Round 3: Données       → Preuves concrètes (benchmarks, patterns, risques)
Round 4: Synthèse      → Position finale li kaydkhol fiha ga3 les objections
Round 5: Scoring       → Score automatique + décision dial l'utilisateur
```
