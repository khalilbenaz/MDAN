MDAN (**M**ulti-**D**omain **A**I **N**etwork) is an AI-driven development platform with specialized agents for FinTech, DevOps/Azure, and Database Optimization. Built on the Better Agents framework, MDAN helps you build software through the whole process from ideation and planning all the way through agentic implementation. It provides specialized AI agents, guided workflows, and intelligent planning that adapts to your project's complexity, whether you're fixing a bug or building an enterprise platform.

If you're comfortable working with AI coding assistants like Claude, Cursor, or GitHub Copilot, you're ready to get started.

:::note[🚀 MDAN v1.0.0 is Here!]
29 specialized agents, 3 domain packs (FinTech, DevOps/Azure, DB Optimization), and complete Better Agents framework integration. **[Check out the GitHub Repository →](https://github.com/khalilbenaz/MDANV2)**
:::

## Extend and Customize

Want to expand MDAN with your own agents, workflows, or modules? The **[Better Agents](https://github.com/langwatch/better-agents)** framework provides the foundation for creating custom extensions, whether you're adding new capabilities to MDAN or building entirely new modules from scratch.

## What You'll Need

MDAN works with any AI coding assistant that supports custom system prompts or project context. Popular options include:

- **[Claude Code](https://code.claude.com)** — Anthropic's CLI tool (recommended)
- **[Cursor](https://cursor.sh)** — AI-first code editor
- **[Codex CLI](https://github.com/openai/codex)** — OpenAI's terminal coding agent

You should be comfortable with basic software development concepts like version control, project structure, and agile workflows. No prior experience with BMad-style agent systems is required—that's what these docs are for.
